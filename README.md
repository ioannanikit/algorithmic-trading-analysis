Algorithmic Trading & Portfolio Optimization
**Project Description**
This project represents a comprehensive approach to algorithmic trading, combining machine learning techniques and modern portfolio theory. The goal is to create a system that analyzes market data (NASDAQ stocks), predicts trends, and optimizes asset allocation to achieve a targeted return while minimizing risk.

**Key Features** 
- Technical Analysis: Calculation of simple moving averages (SMA) and price-to-earnings ratios (P/E Ratio) for stock selection.

- Machine Learning: Use of algorithms to predict price movements and make decisions on entering or exiting positions (Long).

- Portfolio Optimization: Application of Markowitz’s Mean-Variance Optimization to determine optimal weightings per stock.

- Live Integration: Integration with the Alpaca API for real-time trading (paper trading).

- Market Comparison: Benchmarking portfolio performance against the S&P 500 index.

**Technologies & Libraries**
- Python: Programming language.

- Pandas & NumPy: Processing and managing financial data.

- yfinance: Retrieving live market data.

- Matplotlib: Visualizing data and performance (P&L graphs).

- SciPy (optimize): Implementing Markowitz optimization algorithms.
